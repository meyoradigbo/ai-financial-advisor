# ai-financial-advisor
# AI Financial Advisor – Secure Multi‑Agent Prototype

A **production‑style AI financial advisor** built with **FastAPI**, **React**, **SQLite**, and **modular AI agents**.  
The system provides budgeting, cash‑flow forecasting, risk analysis, explainable recommendations, and ML‑driven insights, while prioritising **security, ethics, and transparency**.

---

## 🚀 Key Features

### ✅ Backend (FastAPI + Python)
- Modular **multi‑agent architecture**
- Secure **JWT authentication**
- Provider‑agnostic **open‑banking integration layer**
- ML modules for:
  - Overspending prediction
  - Transaction anomaly / fraud detection
- Advanced **workflow orchestration** with:
  - Failure isolation
  - Audit logging
  - Performance timing
- Explainable AI outputs
- Ethics validation for recommendations
- SQLite database (easy upgrade to PostgreSQL)

### ✅ Frontend (React)
- Modern **financial dashboard**
- Summary cards (income, expenses, savings, risk)
- Interactive charts (pie & line)
- Transaction table with search & filtering
- Financial scenario simulator
- Expandable AI explanation panels
- Risk alerts & savings goal progress
- Dark mode + responsive layout
- Loading skeletons & error states
